# <Trip Eats>

## サイト概要
<旅先での飲食店の検索ができるサイト。>

### サイトテーマ
<飲食店検索。>

### テーマを選んだ理由
<友人と旅行を計画する際、どこのお店で食事するかを考えることに1番時間を使うので、<br>
旅先でのおすすめの飲食店を検索できる、また良かったお店を投稿できるサイトを作りたいと思いました。>

### ターゲットユーザ
<旅行先での計画を考えている人・また旅行先での飲食店を探している人。>

### 主な利用シーン
<旅行を計画している人や旅先の人がスムーズに人気の飲食店や目的の飲食店を探したい時。>
